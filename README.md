# Algorithm Analysis and Complexity Project

Bu repository, **Algoritma Analizi ve Karmaşıklığı** dersi kapsamında verilen proje ödevi için hazırlanmıştır.  
Projede, ders slaytlarında yer alan algoritmalar **C++ (C++11)** kullanılarak uygulanmıştır.

## Proje İçeriği

### 1. Maximum Subsequence Problemi

- Brute Force – O(n³)
- İyileştirilmiş Brute Force – O(n²)
- Divide and Conquer – O(n log n)
- Kadane Algoritması – O(n)

### 2. Rekürsif Algoritmalar

- 1 + 2 + … + N
- Dizi Toplamı
- aⁿ
- Fibonacci
- Tower of Hanoi
- Recursive Digit Sum

### 3. Arama Algoritmaları

- İleri Yönlü Lineer Arama
- Geri Yönlü Lineer Arama
- İkili Arama

### 4. Sıralama Algoritmaları

- Selection Sort
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- External Sorting (teorik)

> Tüm sıralama algoritmaları için çalışma süresi ölçümü **chrono** kullanılarak yapılmıştır.

### 5. Heap Yapıları

- Binary Heap
- findMin, insertHeap, deleteMin
- buildHeap
- decreaseKey, increaseKey, deleteKey
- Heap birleştirme (Merge)

### 6. Greedy Algoritmalar

- İş Zamanlama Problemi
- Aktivite Zamanlama Problemi
- Huffman Kodlama

### 7. Dinamik Programlama

- Bottom-Up DP
- Minimum Maliyet Yol Problemi (Top-Down)
- Sırt Çantası Problemi (0-1 Knapsack)

> Her dinamik programlama problemi için DP tablosu açık şekilde gösterilmiştir.

## Notlar

- Tüm algoritmalar ders slaytları temel alınarak uygulanmıştır.
- STL kütüphanelerinin hazır algoritmaları kullanılmamıştır.
- Proje, eğitim amaçlı hazırlanmıştır.
